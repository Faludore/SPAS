# Generated results

This directory is reserved for artifacts generated by the experiment scripts.

The following subdirectories are created automatically:

```text
section_4_2/
section_4_3/
section_4_4/
```

They may contain:

- CSV tables;
- PNG figures;
- JSON configuration and summary files;
- classification reports;
- text summaries.

Generated artifacts are excluded from version control by default, while this file preserves the `results` directory in the repository.
